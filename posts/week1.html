The course begins with exploring the fundamentals of the Internet, the Web, HTTP, and the structure and common elements of HTML. HTML is simply a markup language for instructing a browser how to display content to the screen. Elements to be displayed can be nested into child-parent hierarchies and their attributes can be set - inline, or through CSS - to achieve a desired layout and appearance.
